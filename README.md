# SupplyFlow Documentation

This directory contains all documentation and resources for the SupplyFlow project.

## Contents

- **`FEATURES.md`** - Standalone markdown document with features (viewable directly on GitHub)
- **`Untitled-2.html`** - Mobile-friendly HTML version of the features document
- **`docs/`** - GitHub Pages documentation site (Jekyll-based)
- **`README-FEATURES.md`** - Setup instructions for GitHub Pages

## Quick Access

### View on GitHub
Simply push this directory to a GitHub repository and view `FEATURES.md` directly.

### GitHub Pages Site
Follow the instructions in `README-FEATURES.md` to set up a full documentation site.

### Local HTML
Open `Untitled-2.html` in any web browser for a mobile-friendly view.
